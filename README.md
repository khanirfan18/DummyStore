# DummyStore

A minimalist e-commerce application built to understand **Supabase authentication** and real-time database interactions.

## Quick Start

```bash
npm install
npm start
```

## Tech Stack

⚛️ **React** — UI framework  
⚡ **Vite** — Build tool  
🔐 **Supabase** — Auth & database  
🛣️ **React Router** — Navigation  
🎨 **React Icons** — Icon library
